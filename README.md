# Neon Doom Runner

A cyberpunk Doom-style 2.5D browser shooter built with plain HTML5 canvas and JavaScript.

## Play Online

After GitHub Pages deploy completes, the game is available at:

- https://allthingssecurity.github.io/doom/

## Controls

- Move: `WASD` or arrow keys
- Turn: mouse or `Q`/`E`
- Shoot: click or `Space`
- Fullscreen: `F`
- Restart campaign: `R`

## Local Run

Serve this folder with any static HTTP server, for example:

```bash
python3 -m http.server 6555
```

Then open:

- http://127.0.0.1:6555
